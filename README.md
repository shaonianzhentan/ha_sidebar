# ha_sidebar
在HA里使用的侧边栏管理


# 使用方式

```

ha_sidebar:

```

```
# 完整配置
ha_sidebar:
  sidebar_title: 侧边栏管理
  sidebar_icon: mdi:xbox-controller-menu
  tabs_name: 二级菜单
  tabs_icon: mdi:format-list-numbered

```
